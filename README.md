## <a name="install"></a>Install

**node.js** :

```bash
$ git clone git@github.com:ccamensuli/nodefony.git

$ cd nodefony

$ make
```

## <a name="start"></a>start
Start the server :
```bash
$ ./nodefony_dev
```
